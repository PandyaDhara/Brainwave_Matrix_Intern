# Brainwave_Matrix_Intern

This tool analyzes URLs to identify potential phishing threats by checking various indicators, such as URL length, use of shortening services, presence of IP addresses, and SSL certificate validity. It also retrieves DNS information and checks domain expiration dates to provide a comprehensive safety assessment.
